## @vscode-use/utils

> WIP: 这个库是为了快速在 vscode 插件中使用一些常用的命令

## Api

- registerCommand 注册指令
- getConfiguration 获取 workspace configuration
- message {type:'info'|'error',message:string,buttons:['ok']} 弹出消息
- openFile 打开某一个文件
